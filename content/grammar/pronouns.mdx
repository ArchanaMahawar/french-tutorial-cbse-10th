## Les Pronoms Personnels COD (Direct Object Pronouns)

Replace the direct object (no preposition before the noun):

| Pronom | Remplace | Exemple |
|--------|----------|---------|
| me (m') | me | Il me regarde. |
| te (t') | you | Je te comprends. |
| le (l') | him / it (masc.) | Je le vois. |
| la (l') | her / it (fem.) | Je la mange. |
| nous | us | Il nous aide. |
| vous | you | Je vous invite. |
| les | them | Je les achète. |

**Placement:** Before the conjugated verb (or before the infinitive).
- Je **la** mange. / Je vais **la** manger.

---

## Les Pronoms COI (Indirect Object Pronouns)

Replace "à + personne":

| Pronom | Remplace |
|--------|----------|
| me (m') | à moi |
| te (t') | à toi |
| lui | à lui / à elle |
| nous | à nous |
| vous | à vous |
| leur | à eux / à elles |

**Exemple:** Je parle à Marie → Je **lui** parle.

---

## Y et EN

**Y** remplace un lieu (à/en/dans/sur + lieu):
- Je vais à Paris → J'**y** vais.

**EN** remplace de + nom (quantité, partitif):
- Je veux du pain → J'**en** veux.
- Il a trois livres → Il **en** a trois.

## Les Pronoms Relatifs

| Pronom | Fonction | Exemple |
|--------|----------|---------|
| **qui** | sujet | L'homme **qui** parle est mon père. |
| **que** | COD | Le livre **que** je lis est bon. |
| **où** | lieu/temps | La ville **où** j'habite est belle. |
| **dont** | de + nom | Le film **dont** je parle est français. |

<ExamNote>
Relative pronouns (qui, que, où, dont) carry 4-6 marks in the grammar section. Practice joining two sentences using the correct pronoun.
</ExamNote>

## Les Pronoms Toniques

Used after prepositions and for emphasis:

| Sujet | Tonique |
|-------|---------|
| je | moi |
| tu | toi |
| il | lui |
| elle | elle |
| nous | nous |
| vous | vous |
| ils | eux |
| elles | elles |

**Usage:** après une préposition (avec moi, chez toi, pour lui), pour l'emphase (Moi, j'aime le chocolat).

## Exercices

1. Remplacez le COD par un pronom : a) Je regarde la télé. b) Il achète les fruits. c) Elle lit le livre.
2. Complétez avec qui, que, où, dont : a) C'est l'ami _____ j'ai parlé. b) La fille _____ chante est belle. c) Le café _____ je vais est fermé.
3. Répondez avec Y ou EN : a) Tu vas au cinéma ? b) Tu veux du lait ?
