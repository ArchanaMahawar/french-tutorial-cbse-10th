## Format du Message

A message (un message) is a short note left for someone. It carries **5 marks** in Section B.

### Structure:

```
MESSAGE

Date : le [date]
Heure : [time]

[Destinataire (To whom)],

[Content — reason, information, instruction in 3-4 sentences]

[Signature (your name)]
```

---

## Caractéristiques (Features)

- Very short: 30-40 words
- Direct and clear
- Includes: date, time, who it's for, the information, your name
- Informal tone (tu/toi for friends/family, vous for formal)

---

## Phrases Utiles

- Je t'informe que... (I'm informing you that...)
- N'oublie pas de... (Don't forget to...)
- Je serai de retour à... (I'll be back at...)
- Appelle-moi quand... (Call me when...)
- Je suis sorti(e) pour... (I've gone out to...)

---

## Modèles

### Modèle 1 — Message à un ami

**Situation:** Vous êtes allé(e) chez votre ami(e) mais il/elle n'était pas là. Laissez un message.

```
MESSAGE

Date : le 10 octobre 2026
Heure : 15h30

Cher Rahul,

Je suis passé(e) chez toi mais tu n'étais pas là.
Je voulais te rendre ton livre de français.
Je reviendrai demain après l'école.
Appelle-moi ce soir.

Anita
```

### Modèle 2 — Message à maman

**Situation:** Vous devez sortir. Laissez un message pour votre mère.

```
MESSAGE

Date : le 5 novembre 2026
Heure : 16h00

Chère Maman,

Je suis sorti(e) pour acheter des fournitures
scolaires. Je serai de retour avant 18h.
Le dîner est dans le frigo.

Arjun
```

<ExamNote>
Always include date, time, greeting, message content, and signature. Keep it between 30-40 words. The message should be clear and complete — the reader should understand exactly what to do.
</ExamNote>

## Modèle 3 — Quelqu'un a téléphoné

**Situation (2021):** Votre sœur n'est pas à la maison. Le dentiste a appelé. Laissez un message.

```
MESSAGE

Date : le 12 mars 2026
Heure : 14h30

Chère Meera,

Le Dr. Sharma a téléphoné. Il a confirmé ton
rendez-vous pour demain à 10h. N'oublie pas
d'apporter ta carte d'assurance.

Ravi
```

4. Le facteur a déposé un colis. Laissez un message pour votre père.
