# Product Overview

A self-study tutorial and assessment platform for CBSE Class 10 French students (Indian board).

## Purpose

Help 10th standard CBSE students learn French independently through chapter-wise tutorials aligned with the prescribed textbook "Entre Jeunes (Class X)" and the Setrite Complete Practice Material workbook, practice grammar, writing skills, and self-assess using CBSE-pattern sample papers.

## Target Users

- Class 10 CBSE board students studying French as a third language (Code 018)
- Students preparing for the annual board examination (80 marks written + 20 marks internal)
- Students using Setrite French - 10 (Complete Practice Material) by Pooja Tyagi

## Key Features

- Chapter-wise tutorials based on "Entre Jeunes" textbook & Setrite workbook
- Board exam focused: only covers lessons prescribed for 2025-26 (Leçons 2-8, 10)
- Grammar tutorials (tenses, pronouns, adjectives, negation, question formation)
- Comprehensive writing section (informal letters, messages, dialogues, fill-in-the-blank compositions)
- CBSE-pattern sample papers with marking scheme and answer keys
- Self-assessment quizzes with instant feedback
- Mobile-friendly responsive design for accessibility
